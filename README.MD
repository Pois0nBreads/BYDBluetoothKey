# BYD老款OBD蓝牙钥匙 控制APP
## 项目简介
- 项目基于秦遥控驾驶APP逆向而来，取消原APP蓝牙扫描功能，直接读取系统已配对蓝牙列表。进入APP从保存的macaddress直接连接，主打一个块。
- （早期AndroidAPP似乎都有这个毛病，app里搜索，但是还要去系统设置里配对，多此一举）
- 设置入口放在ShortCuts里了，所有界面都用单实例打开，进入设置 遥控界面会自动退出。
- 其中命令生成库是BYD的native本地库，暂时还没功夫逆向，直接用现成的。位于com.byd.jnitest.JNI.java (吐槽一下这个叼命名)
- 坏消息是JNI的so文件只有armeabi，可能后续会有兼容性问题。有功夫的老哥可以看看逆向。
